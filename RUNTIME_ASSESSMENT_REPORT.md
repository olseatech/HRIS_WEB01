## 6. POTENTIAL ISSUES & MITIGATIONS

### Issue 1: Passport Photo Field Not Displaying
**Severity**: Medium  
**Cause**: Missing CSRF meta tags in HTML  
**Mitigation**:
- Verify `<meta name="_csrf">` and `<meta name="_csrf_header">` are in the HTML header
- Check that Spring Security is configured to include these in templates

### Issue 2: Date Mask Not Working on Linux
**Severity**: Low  
**Cause**: jQuery not loaded or timezone issues  
**Mitigation**:
- Ensure jQuery is properly included in the page
- Test in browser DevTools console: `typeof jQuery` should return "function"
- Check for JavaScript errors in browser console

### Issue 3: Part-Time Work Experience Not Saved
**Severity**: Low  
**Cause**: Form not submitting `partTime` field  
**Mitigation**:
- Verify `<input ... name="partTime">` is present in HTML form
- Check that WorkExperience model has getter/setter for partTime
- Test form submission and check database value

### Issue 4: GSIS ID Not Displaying on PDS Report
**Severity**: Low  
**Cause**: Parameter added but text field may not be positioned correctly  
**Mitigation**:
- Verify the parameter is properly bound in ReportsController.populateMapReport2025_P1()
- Check that emp.getGsisIdNo() returns non-null value
- Regenerate Jasper reports from JRXML: `mvn jasperreports:jasper`

### Issue 5: File Upload Fails Due to Permissions
**Severity**: Medium  
**Cause**: `/hrisp/uploads` or `/hrisp/tmp` not writable  
**Mitigation**:
- Create directories with proper permissions: `mkdir -p /hrisp/{uploads,tmp}`
- Set ownership: `chown tomcat:tomcat /hrisp/{uploads,tmp}`
- Set permissions: `chmod 755 /hrisp/{uploads,tmp}`
